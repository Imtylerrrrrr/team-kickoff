# templates/ 사용 규칙

- `{{PLACEHOLDER}}`는 인터뷰·모드 결정 결과로 치환한다:
  - `{{LANG}}` → 한글 / English (인터뷰 ③)
  - `{{REVIEW_RULE}}` → 해커톤: "승인 없이 셀프 머지 허용 (PR은 반드시 거친다)" / 정규: "1명 승인 후 머지, 셀프 머지 금지" / 1인: "승인 없이 머지"
  - `{{PROTECTION_NOTE}}` → 브랜치 보호 하드 적용 시: "" (빈 문자열) / 소프트 강등 시: " (설정으로 강제되지 않음 — 규칙으로 지킨다)"
- 사용자의 언어가 한국어가 아니면 템플릿 전체를 그 언어로 번역해 생성한다.
- 마커(`<!-- team-rules:start/end -->`)는 언어와 무관하게 그대로 유지한다 — 재실행 멱등성의 기준점.
- 기존 파일에 삽입할 때의 규칙은 SKILL.md 4단계 표를 따른다 (덮어쓰기 금지).
