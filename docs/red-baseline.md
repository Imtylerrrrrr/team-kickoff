## 베이스라인이 잘한 것 (스킬이 다시 가르칠 필요 없는 것)

- squash-only + GitHub Flow 선택, husky 대신 무의존 `core.hooksPath`
- PR 템플릿에 "diff 전체를 읽고 설명할 수 있다" 검증 항목 (채택 가치 있음)
- CI로 PR 제목 검사 (무료 private에서도 빨간 X — 정규 모드 옵션으로 채택)
